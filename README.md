# NotePad
## 项目说明：这个实验是在NotePad应用上做拓展功能

### 1、基本要求：
NoteList中显示条目增加时间戳显示
添加笔记查询功能（根据标题查询）

### 2、扩展功能：
UI美化、笔记排序、导出笔记
